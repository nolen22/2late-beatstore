# 2Late Beat Store

A modern, luxurious beat selling platform built for trap music producer 2Late.

## 🔧 Tech Stack
- **Frontend**: React + TailwindCSS
- **Backend**: Firebase (auth, Firestore, storage)
- **Payments**: Stripe Checkout (Basic, Premium, Exclusive licenses)
- **Deployment**: Vercel
- **Email**: Formspree
